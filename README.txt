# How to play #######################################
                                                    #
Map Editor :                                        #
                                                    #
- E      : show list of blocks                      #
- I      : show list of items                       #
- S      : save the map                             #
- ESCAPE : return to the main menu                  #
                                                    #
Game :                                              #
                                                    #
- I           : Show inventory                      #
- RIGHT CLICK : Move                                #
- LEFT CLICK  : Pick up and drop items              #
- ESCAPE      : return to the main menu             #
                                                    #
